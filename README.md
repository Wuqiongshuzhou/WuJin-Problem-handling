# 题目数据解析表格工具

一个基于纯前端技术的题目数据解析与可视化工具，支持将Markdown格式的题目文件解析为交互式表格，并提供数据统计、图表展示、相似题目推荐等功能。

## 功能特性

### 核心功能
- **Markdown解析**：自动解析Markdown格式的题目文件，提取题目名、考点等信息
- **多文件支持**：支持同时加载多个Markdown文件，可单独激活/禁用文件
- **交互式表格**：以表格形式展示题目数据，支持排序、选择、复制等操作
- **智能搜索**：支持多种搜索模式（部分匹配或/且、精确匹配），支持关键词高亮

### 数据可视化
- **统计面板**：展示题目总数、考点分布、难度分布、题型分布等统计数据
- **图表展示**：支持条形图、饼图、环形图、折线图、雷达图等多种图表类型
- **数据筛选**：图表支持滑动条筛选，动态调整显示范围

### 特色功能
- **相似题目推荐**：基于考点相似度智能推荐相似题目
- **批量复制**：支持批量复制Obsidian格式的题目链接
- **深色模式**：支持浅色/深色主题切换
- **水印保护**：可配置页面水印
- **鼠标轨迹特效**：炫酷的鼠标跟随动画效果

## 使用方法

### 快速开始
1. 直接在浏览器中打开 `解析为表格-完整版.html` 文件
2. 将Markdown格式的题目文件拖拽到页面上的拖放区域
3. 程序会自动解析并展示题目数据

### Markdown文件格式要求
题目文件需要遵循以下Markdown格式：

```markdown
#### 题目标题
##### （考点：）【考点1，考点2，考点3，难度低，选择，历史】
题目内容...
```

- 四级标题（`####`）为题目名
- 五级标题（`#####`）中包含考点信息，格式为`【考点1，考点2，...】`

### 操作说明

| 操作 | 说明 |
|------|------|
| 拖拽文件 | 将Markdown文件拖拽到页面顶部区域 |
| 点击行 | 选中/取消选中该行 |
| 右键点击行 | 复制该题目的Obsidian链接 |
| 点击文件标签 | 激活/禁用该文件 |
| 点击标签上的× | 移除该文件 |

### 搜索模式
- **部分匹配（或）**：任一关键词匹配即可
- **部分匹配（且）**：所有关键词都必须匹配
- **精确匹配（且）**：关键词必须完全匹配考点

## 特色亮点

1. **零依赖部署**：单HTML文件，无需安装任何依赖，直接浏览器打开即可使用
2. **数据本地处理**：所有数据处理均在本地完成，保护数据隐私
3. **响应式设计**：适配不同屏幕尺寸
4. **用户体验优化**：支持快捷操作、视觉反馈、平滑动画等
